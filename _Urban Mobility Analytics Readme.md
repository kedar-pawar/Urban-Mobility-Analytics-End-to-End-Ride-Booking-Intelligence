> Project Overview

This project presents an end-to-end urban mobility analytics solution
built using Power BI, focused on analysingride-booking data to uncover
demand patterns, operational efficiency, revenue drivers, cancellations,
andcustomer & driver experience.

The dashboard is designed to simulate a real-world ride-hailing platform
(similar to Ola/Uber) and demonstrates how data-driven insights can
support business and operational decisions.
