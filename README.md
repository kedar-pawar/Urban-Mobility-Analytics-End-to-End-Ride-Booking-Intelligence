Urban Mobility Analytics is an end-to-end data analytics project that analyzes ride-booking operations for an urban mobility platform. The project integrates Power BI, Python, SQL, and MySQL to perform data extraction, exploratory data analysis (EDA), KPI calculation, and interactive dashboard reporting. The objective is to transform raw booking data into actionable business insights related to operations, revenue, cancellations, and customer & driver experience.

ðŸ”¹ Problem Statement

Urban ride-booking platforms face challenges such as high cancellation rates, uneven fleet utilization, revenue leakage, and fluctuating customer satisfaction. This project addresses these challenges by answering:

What is the overall booking success and cancellation rate?

Which vehicle types perform best operationally and financially?

What factors contribute most to ride cancellations?

How do customer and driver ratings vary across vehicle categories?
